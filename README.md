# 🧠 AI Notes Manager

A modern Notes Manager built using Streamlit with a clean UI and AI-ready features.

## 🚀 Features
- Add, View, Search, Delete Notes
- Dashboard with analytics
- Excel-based storage
- AI-ready (Summarize, Rewrite - coming soon)

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
